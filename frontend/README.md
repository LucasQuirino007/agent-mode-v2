# Frontend — Portal de Consulta de Documentos

Aplicação frontend em **React 18 + Vite 5 + Tailwind CSS 3**.

## Requisitos

- Node.js 20 LTS+
- npm 9+

## Instalação

1. Entre na pasta `frontend`
2. Instale dependências com `npm install`

## Desenvolvimento

1. Garanta que o backend esteja rodando em `http://localhost:3001`
2. Execute `npm run dev`
3. Acesse `http://localhost:5173`

## Variáveis de ambiente

Crie (ou ajuste) o arquivo `.env` na pasta `frontend/`:

`VITE_API_URL=http://localhost:3001`

## Build

Execute `npm run build` para gerar a pasta `dist/`.
